Matrix-Lab-Experiments 矩阵计算实践作业总览

二、各次实践作业核心内容总览

本仓库将按作业顺序，依次存放各次实践作业的全部相关文件，以下为各作业核心要求（后续作业将持续更新）：

实践作业（一）（满分25分）

提交时间：第四周周日前

题目要求：

1. （10分）结合教材算法 1.3.1 和算法 1.3.2，编写平方根法（Cholesky分解）和改进的平方根法的Matlab程序，求解对称正定线性方程组Ax=b：

- 任务1：系数矩阵A为100阶对称正定随机矩阵，右端向量b随机选取；

- 任务2：系数矩阵A为40阶Hilbert矩阵（元素满足 \(a*{ij}=\frac{1}{i+j-1}\)），右端向量b的分量满足 \(b_i=\sum*{j=1}^n \frac{1}{i+j-1}\)。

2. （15分）编写教材算法 2.5.1（估计矩阵的1范数：优化法）的Matlab程序，计算Hilbert矩阵的条件数（禁止使用Matlab内置求条件数命令，需严格按照算法伪代码编写）。

完成要求：

- 附上完整Matlab程序；

- 给出程序运行的详细结果；

- 提供详细的图表数据分析，完成完整实验报告。

实践作业（二）及后续作业

后续作业将根据课程进度持续更新，更新内容将包含：题目要求、Matlab程序编写、运行结果、数据分析及实验报告，保持与作业（一）一致的排版规范和文件结构，便于统一管理和查阅。
